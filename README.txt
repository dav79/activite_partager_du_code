Objetif du projet :
Le but de ce petit projet est d'envoyer des informations via un formulaire prevenant d'une page.jsp, puis de récupérer les paramètres de l'url envoyées 
par la méthode post et de les afficher à l'écran.

Les fichiers du projet :
Formulaire jsp
Formulaire de saisie d'informations

Resultats.jsp
Récupérer les valeurs des paramètres envoyer par la méthode post
